Etiquetas HTML Fundamentales

A continuación encontrarás explicaciones y ejemplos de algunas etiquetas esenciales en HTML.

Tabla de Contenidos

  • Headers
  • Enlaces

1. Etiquetas <h1> y <h2> — Estructura

Las etiquetas de encabezado permiten organizar la estructura jerárquica de una página web.


<h1>Bienvenido a mi página</h1>
<h2>Sobre mí</h2>

Más información: W3Schools — Encabezados HTML

2. Etiqueta <a> — Enlaces

Crea un hipervínculo a otra web, documento o sección interna.


<a href="https://www.google.com" title="Ir a Google">
  Ir a Google
</a>

Más información: W3Schools — Etiqueta <a>

3. Etiqueta <img> — Imágenes

Inserta imágenes en la página. No tiene etiqueta de cierre.


<img src="imagenes/logo.png" alt="Logo de ejemplo" width="200">

Más información: W3Schools — Etiqueta <img>

4. Etiqueta <form> — Formularios

Permiten enviar datos al servidor.


<form action="procesar.php" method="post">
  <label>Nombre:</label>
  <input type="text" name="nombre">

  <label>Email:</label>
  <input type="email" name="email">

  <input type="submit" value="Enviar">
</form>

Más información: W3Schools — Formularios HTML

5. Etiqueta <table> — Tablas

Organiza datos en filas y columnas.


<table border="1">
  <tr>
    <th>Producto</th>
    <th>Precio</th>
  </tr>
  <tr>
    <td>Chocolate</td>
    <td>2€</td>
  </tr>
</table>

Más información: W3Schools — Tablas HTML
