1. Color y Fondo

Define colores de texto y de fondo.


p {
  color: red;
  background-color: yellow;
}

Más información: W3Schools — Colores CSS

2. Tipografía

Controla el tipo de letra, tamaño y estilo.


h1 {
  font-family: Arial, sans-serif;
  font-size: 24px;
  font-weight: bold;
}

Más información: W3Schools — Tipografía CSS

3. Márgenes y Padding

Define espacios externos e internos de los elementos.


div {
  margin: 20px;
  padding: 10px;
}

Más información: W3Schools — Márgenes y Padding

4. Bordes

Permite definir bordes de los elementos.


div {
  border: 2px solid black;
  border-radius: 10px;
}

Más información: W3Schools — Bordes CSS

5. Flexbox

Diseño flexible para organizar elementos en filas o columnas.


.container {
  display: flex;
  justify-content: center;
  align-items: center;
}

Más información: W3Schools — Flexbox CSS
